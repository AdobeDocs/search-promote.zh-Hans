# 关于主页{#about-home}

您可以使用“主页”视图各种报表构件，以便快速概述Search&amp;Promote帐户。

您可以拖放报表构件来重新排列主页，而不会影响其他是帐户成员的用户。 您还可以从主页中删除报表构件，也可以使用添加内容下拉列表将其添加回来。
