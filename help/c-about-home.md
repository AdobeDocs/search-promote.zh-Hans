# 关于主页{#about-home}

您可以使用“主页”查看各种报告构件，这些构件可以快速概述您的Search&amp;Promote帐户。

您可以拖放报表构件来重新排列主页，而不会影响帐户成员的其他用户。 您还可以从主页中删除报表构件，也可以使用添加内容下拉列表将其添加回来。
